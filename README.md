# MeetMind AI 🚀

AI-powered meeting intelligence tool that converts transcripts into structured insights.

## Features
- Summary, Decisions, Action Items
- Checklist UI
- Keyword highlighting
- Word count
- Sample input & clear button
- Progress bar
- Download output
- Modern UI

## Tech Stack
- Python
- Streamlit
- Groq API (Free)

## Run

pip install -r requirements.txt  
streamlit run app.py
